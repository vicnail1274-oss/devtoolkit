## Full Description
DevToolkit is a collection of 30+ free online developer tools designed for speed and privacy.

Every tool runs entirely in your browser — no data is ever sent to a server, no account is required, and there's no tracking. Just open and use.

**What's included:**
- JSON Formatter & Validator
- JSON Schema Generator
- API Tester (Postman-lite in browser)
- Cron Expression Generator
- Regex Tester
- Base64/URL Encoder
- JWT Decoder
- Hash Generator
- UUID Generator
- Diff Checker
- Token Counter (GPT/Claude/Llama)
- Color Converter, CSS Minifier, and many more

**Why developers love it:**
- Zero friction: no signup, no install, no cookies
- Blazing fast: built with Astro + Preact, sub-second load times
- Privacy-first: everything is client-side JavaScript
- Mobile-friendly: works on any device
- SEO-optimized: each tool has its own page for easy bookmarking

**Pro tier ($9.99/mo)** adds 3 AI-powered tools:
- AI Code Review
- AI Doc Generator
- AI SQL Builder

Built by a solo maker who needed fast, no-nonsense dev tools.
